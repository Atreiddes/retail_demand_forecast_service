Зачем нужен интервал? (пояснение)

Одна средняя цифра не говорит о риске. Интервал показывает разброс, и заказ можно делать под свою цену ошибки:

  • Скоропорт (списывается) - заказывать ближе к нижней границе P10, чтобы не выбрасывать лишнее.
  • Ходовой дефицитный товар - ближе к верхней границе P90, чтобы не терять продажи.
  • Узкий интервал - можно довериться прогнозу; широкий - проверить руками.

Так интервал напрямую помогает не допускать перестока и недостока.

Сводка по агрегатам

Суммарный прогноз P50 за горизонт по уровням: всего, по регионам, магазинам и отделам. Модель прогнозирует на уровне товар-магазин, уровни выше - суммы этих прогнозов снизу вверх. На агрегатах модель точнее всего (см. WRMSSE ниже). Ширина интервала агрегата - сумма интервалов рядов: это верхняя, осторожная оценка неопределённости (реальный разброс суммы уже).

Качество модели

Бэктест walk-forward по нескольким окнам, WRMSSE по уровням иерархии M5. Оценка модели, общая для всех прогонов, а не точность этого прогона. WRMSSE меньше 1 - лучше наивного прогноза (повтор прошлой недели), больше 1 - хуже. Зелёный хорошо, красный - на уровне наива или хуже.

Что означают уровни L1-L12? (пояснение)

Прогноз проверяется на разных уровнях агрегации: сверху общие итоги (их прогнозировать легче), внизу отдельный товар в конкретном магазине (сложнее всего).

  • L1 - всего продаж по всем магазинам и товарам
  • L2 - по штату (региону)
  • L3 - по магазину
  • L4 - по категории (у нас одна FOODS, поэтому совпадает с L1)
  • L5 - по отделу внутри категории
  • L6 - штат по категории (на срезе FOODS совпадает с L2)
  • L7 - штат по отделу
  • L8 - магазин по категории (совпадает с L3)
  • L9 - магазин по отделу
  • L10 - по товару (сумма по всем магазинам)
  • L11 - товар по штату
  • L12 - конкретный товар в конкретном магазине (самый нижний уровень)

Верхние уровни у нас 0.2-0.7 - заметно лучше наива. Нижний L12 около 1: спрос по одному товару в магазине прерывистый, точнее наива предсказать почти нельзя. Поэтому ценность в агрегатах (категория, магазин, регион) и в интервале P10-P90, а не в точечном прогнозе одного SKU.
